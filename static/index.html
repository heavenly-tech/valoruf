Valor UF

Datos obtenidos usando la API de CMF Chile api.cmfchile.cl.

Funciones API:

  • /api/uf/<fecha> - Obtiene el valor de la UF para una fecha específica (formato AAAA-MM-DD).
  • /api/uf/<fecha_inicio>/<fecha_fin> - Obtiene los valores de la UF para un rango de fechas.
  • /api/uf/cached - Obtiene todos los valores almacenados en el caché.
  • Por defecto, todos los endpoints devuelven formato JSON.
  • Puedes usar ?format=json o ?format=csv en los endpoints de rango de fecha y de caché.
  • El endpoint de fecha única soporta ?format=json y ?format=raw (texto plano).
